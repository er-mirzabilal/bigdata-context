The Open Pragmatics Standard is to help all data practitioners build a performant data context layers that work with AI. It’s a toolkit that includes:

  • The standard — an open, vendor-neutral vocabulary for enterprise context: metrics, entities, policies, decisions, and the tribal knowledge that never made it into a schema.
  • Evaluation tools — open benchmarks and tests to measure how legible your data actually is to AI, and whether a change made it better.
  • Reference architectures — proven blueprints for standing up a pragmatic context layer on the stack you already run.
  • Playbooks — the patterns, step by step, for the work: migrations, metric harmonization, keeping context current.
  • Ecosystem — of likeminded contributors working on the mission.

Published in the open, so your agents are never locked to anyone’s implementation — including ours.
